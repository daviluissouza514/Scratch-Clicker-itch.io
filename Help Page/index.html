Frequently Asked Questions

My Save button isn't working!
Ensure you are not in "Incognito/Private" mode, as browsers often block localStorage in those modes. Also, make sure you have allowed cookies for this site.
How do Save Codes work?
When you click Save, the game generates a unique string of text. Copy and save this text in a Notepad file. You can use the Load button to paste it back later!
Does the game click while I'm away?
Only if you have purchased Passive Income upgrades. The game must remain open in a tab to collect gold unless "Offline Earnings" is unlocked.
What is the maximum level?
There is no hard limit! However, costs will increase exponentially. Reaching 1 Quadrillion clicks is the ultimate community goal.
